API Documentation

Security Features
  • No client-side caching for enhanced security
  • Server appears as PHP 4.2.0 (security through obscurity)
POST /api/books

Add a new book to the collection

Request Body: 
{
  "title": "string" // required
}
Response: 
{
  "_id": "string",
  "title": "string"
}
GET /api/books

Retrieve all books in the collection

Response: 
[{
  "_id": "string",
  "title": "string",
  "commentcount": number
}]
GET /api/books/{_id}

Retrieve a specific book by ID

Response: 
{
  "_id": "string",
  "title": "string",
  "comments": ["string"]
}
Returns "no book exists" if ID not found
POST /api/books/{_id}

Add a comment to a specific book

Request Body: 
{
  "comment": "string" // required
}
Response: 
{
  "_id": "string",
  "title": "string",
  "comments": ["string"]
}
DELETE /api/books/{_id}

Delete a specific book

Response: 
"delete successful"
DELETE /api/books

Delete all books in the collection

Response: 
"complete delete successful"
